# Digital Employee Management & Performance Optimization (DEMPO)

##  Project Overview

Digital Employee Management & Performance Optimization (DEMPO) is a Final Year Project designed to digitalize and automate employee management processes within organizations. The system provides a centralized platform for managing employees, attendance, leave requests, performance evaluations, and reporting.

The main goal of this system is to replace traditional manual HR processes with a smart, efficient, and scalable digital solution.

---

##  Problem Statement

Most organizations still rely on manual or semi-digital systems for HR management, which leads to:

- Data inconsistency  
- Time-consuming processes  
- Difficulty in tracking employee performance  
- Lack of centralized data management  
- Inefficient reporting system  

DEMPO solves these problems by providing an integrated digital solution.

---

##  Objectives

- Digitize employee record management  
- Automate attendance tracking  
- Simplify leave request and approval process  
- Enable performance evaluation system  
- Provide real-time reporting and analytics  
- Improve HR efficiency and decision-making  

---

##  Key Features

###  Employee Module
- Employee registration and profile management  
- View personal details and performance history  
- Submit leave requests  
- View attendance records  

###  Admin Module
- Add, update, and delete employees  
- Manage attendance records  
- Approve/reject leave requests  
- Evaluate employee performance  
- Generate reports and analytics dashboard  

###  Authentication System
- Secure login system  
- Role-based access control (Admin / Employee)  
- Password protection and session management  

---
