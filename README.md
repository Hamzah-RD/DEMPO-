# Digital Employee Management & Performance Optimization (DEMPO)

##  Project Overview

DEMPO is a Final Year Project designed to digitalize employee management in organizations. It provides a centralized system to manage employees, attendance, leave requests, and performance evaluation.

The system replaces manual HR processes with a simple, fast, and efficient digital solution.

---

##  Problem Statement

Traditional HR systems face issues such as:

- Manual record keeping  
- Time-consuming processes  
- Difficulty in tracking employee performance  
- Lack of centralized data  

This project solves these problems through automation and digital management.

---

##  Objectives

- Manage employee records digitally  
- Automate attendance system  
- Handle leave requests efficiently  
- Track employee performance  
- Improve HR decision-making
